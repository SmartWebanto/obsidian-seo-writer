# Tone of Voice

## Voice spectrum

| Dimension | 1 | 2 | 3 | 4 | 5 | Position |
|-----------|---|---|---|---|---|----------|
| Formal ↔ Casual | | | | X | | 4 — friendly, like a barista explaining |
| Serious ↔ Playful | | | X | | | 3 — mostly helpful, occasional coffee pun |
| Technical ↔ Simple | | | X | | | 3 — explain the science, keep it accessible |
| Reserved ↔ Enthusiastic | | | | X | | 4 — we genuinely love this stuff |
| Corporate ↔ Human | | | | | X | 5 — real people, real opinions |

## Writing rules

| Rule | Setting |
|------|---------|
| Sentence length | Under 20 words average |
| Paragraph length | 2-3 sentences max |
| Jargon policy | Use coffee terms but explain on first use (e.g., "extraction — how much flavor the water pulls from the grounds") |
| Contractions | Always (we're, it's, don't) |
| First person | "We" for Brew & Bean opinions, "you" for the reader |
| Emoji use | Never in articles |
| CTA style | Helpful nudge, not hard sell ("Worth trying if..." not "Buy now!") |
| Voice | Active. "We tested 12 grinders" not "12 grinders were tested" |

## Page-type adjustments

| Page type | Tone shift | Notes |
|-----------|-----------|-------|
| Blog post | Slightly more conversational | Share personal testing anecdotes |
| Landing page | More confident, benefit-led | Focus on the recommendation |
| Product page | Detail-oriented, specs-forward | Trust the reader to decide |
| FAQ | Simpler, direct answers | No fluff |

## Words to use / avoid

| Use | Avoid | Why |
|-----|-------|-----|
| "we tested" | "experts say" | We ARE the testers |
| "worth it" | "game-changer" | Honest > hype |
| "solid choice" | "must-have" | Recommend, don't pressure |
| "in our experience" | "objectively the best" | Stay humble |
| "budget-friendly" | "cheap" | Respect the reader's money |

## Key messages

1. You don't need expensive gear to make great coffee — you need the right gear for YOUR routine.
2. We test everything ourselves, in real kitchens, with real coffee. No sponsored placements.
3. Great coffee at home is a skill anyone can learn. We'll show you how.

## Channel adjustments — Web / SEO

| Context | Tone shift | Notes |
|---------|-----------|-------|
| Blog post | Conversational, anecdote-rich | "When we first tried this grinder..." |
| Product page | Specs + opinion balanced | Lead with what matters, then numbers |
| Landing page | Confident recommendation | "After testing 12 options, here's what we'd buy" |
| Category page | Scannable, structured | Use tables, comparison grids |
| Meta description | Benefit-first, action verb | "Find the right grinder for your brew style" |
